Metro Facts - Employee Performance App

This project is a Tkinter application that allows employees and supervisors of Metro Facts to access and view performance metrics.

Features:

Employees can view their individual metrics for the month, including tasks completed, calls taken, products sold, and task errors.
Employees can see if they earned a bonus for the month (placeholder function).
Supervisors can view the sum of all employee metrics for the month (functionality not implemented yet).
User Access:

Both employees and supervisors require a unique access ID to view any information within the application.
A pre-defined list of employees with their access IDs is included in the code.
Running the Application:

Clone the repository or download the project files.
Install the required libraries: tkinter, PIL, and any others listed in requirements.txt (if applicable).
Ensure image paths in the code (c:/gui/sgdiamond.ico and c:/metro facts/fact python image.png) are adjusted to point to the correct locations on your system.
Run the main.py script (or equivalent).
